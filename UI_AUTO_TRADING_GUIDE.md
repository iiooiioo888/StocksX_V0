# StocksX 自動交易 UI 使用指南

## 📋 概述

已為您創建**完整的圖形化用戶界面**，包含 5 個主要功能模組：

| 模組 | 文件 | 功能 |
|------|------|------|
| **儀表板** | `dashboard.py` | 整體績效、持倉總覽、策略狀態 |
| **持倉監控** | `position_monitor.py` | 即時持倉、損益追蹤、操作按鈕 |
| **策略配置** | `strategy_config.py` | 交易所設定、策略選擇、參數配置 |
| **風險管理** | `risk_manager_ui.py` | 風險計算器、倉位計算、停損/停利 |
| **交易日誌** | `trade_log.py` | 交易記錄、績效統計、圖表分析 |

### 1. 📊 儀表板

**路徑**: 自動交易 → 📊 儀表板

**功能**:
- ✅ 整體績效指標（總損益、總權益、勝率）
- ✅ 持倉總覽表格
- ✅ 策略狀態管理
- ✅ 快速操作按鈕

**指標說明**:
| 指標 | 說明 |
|------|------|
| 總損益 | 所有交易的累計損益 |

**快速操作**:
- 🛑 緊急停止所有交易
- 📊 生成每日報告
- 🔄 重新整理
- 📥 匯出交易記錄

### 3. 🤖 策略配置

**路徑**: 自動交易 → 🤖 策略配置

**功能**:
- ✅ 交易所設定
- ✅ 策略訂閱（支援多策略）
- ✅ 參數配置
- ✅ 風險設定
- ✅ 配置預覽

#### 步驟 1: 交易所設定

| 欄位 | 說明 | 範例 |
|------|------|------|
| 交易所 | 選擇交易所 | Binance、OKX、Bybit |
| 測試網絡 | 是否使用測試網絡 | ✅ 建議勾選 |
| API Key | 交易所 API Key | `your_api_key` |
| API Secret | 交易所 API Secret | `your_api_secret` |

#### 步驟 2: 策略訂閱

**支援的策略**:

| 分類 | 策略 | 說明 |
|------|------|------|
| **趨勢** | `sma_cross` | 雙均線交叉 |
| | `ema_cross` | 指數均線交叉 |
| | `macd_cross` | MACD 交叉 |
| | `supertrend` | 超級趨勢 |
| | `adx_trend` | ADX 趨勢強度 |
| | `parabolic_sar` | 拋物線 SAR |
| **擺盪** | `rsi_signal` | RSI 超買超賣 |
| | `stochastic` | KD 隨機指標 |
| | `williams_r` | 威廉指標 |
| | `bollinger_signal` | 布林帶 |
| **突破** | `donchian_channel` | 唐奇安通道 |
| | `dual_thrust` | 雙推力突破 |
| **其他** | `vwap_reversion` | VWAP 回歸 |
| | `ichimoku` | 一目均衡表 |
| | `buy_and_hold` | 買入持有 |

**訂閱設定**:
- 交易對：如 `BTC/USDT:USDT`
- 策略：從下拉選單選擇
- 時間框架：1m/5m/15m/1h/4h/1d
- 啟用：勾選是否啟用此策略

#### 步驟 3: 策略參數

每種策略有不同的參數：

**SMA Cross**:
- 快線週期：5
- 慢線週期：20

**MACD Cross**:
- 快線：12
- 慢線：26
- 信號線：9

**RSI Signal**:
- 週期：14
- 超賣線：30
- 超買線：70

**Bollinger Signal**:
- 週期：20
- 標準差：2.0

#### 步驟 4: 風險管理

| 參數 | 預設值 | 說明 |
|------|--------|------|
| 每筆風險 | 2% | 每筆交易承擔的風險 |
| 停損 | 2% | 停損百分比 |
| 停利 | 4% | 停利百分比 |
| 最大持倉數 | 3 | 同時持倉上限 |
| 槓桿倍數 | 1x | 槓桿倍數 |
| 每日最大虧損 | 5% | 每日虧損上限 |

**進階選項**:
- 倉位計算方法：固定比例/凱利公式/固定金額
- 移動停損：啟用後自動追蹤獲利
- 最大回撤：總資金回撤上限
- 任務結束時平倉：停止時自動平倉

#### 步驟 5: 資金設定

- 初始資金 (USDT)：建議至少 $1,000

**按鈕**:
- 💾 儲存策略配置：儲存配置但不啟動
- ▶️ 立即啟動：儲存並啟動自動交易

### 4. 🛡️ 風險管理工具

**路徑**: 自動交易 → 🛡️ 風險管理

**功能分頁**:

#### 📊 風險計算器

計算風險報酬比和期望值：

**輸入**:
- 進場價
- 停損價
- 停利價
- 預估勝率

**輸出**:
- 風險 (%)
- 報酬 (%)
- 風險報酬比
- 期望值
- 分析建議

#### 💰 倉位計算器

三種計算方法：

**1. 固定比例 (Fixed Fraction)**
```
倉位 = (權益 × 風險%) / (進場價 - 停損價)
```

**2. 凱利公式 (Kelly)**
```
f* = (p × b - q) / b
```
- p = 勝率
- q = 1 - p
- b = 盈虧比

**3. 固定金額 (Fixed Amount)**
```
倉位 = 權益 × 固定比例 / 進場價
```

#### 🛑 停損/停利計算器

**功能**:
- 計算停損/停利價格
- 視覺化示意圖
- 移動停損設定

**多頭範例**:
```
進場價：$50,000
停損：2% → $49,000
停利：4% → $52,000
```

**空頭範例**:
```
進場價：$50,000
停損：2% → $51,000
停利：4% → $48,000
```

#### 📈 風險分析

分析交易系統的風險特徵：

**輸入統計**:
- 總交易數
- 勝率
- 平均獲利
- 平均虧損

**輸出指標**:
- 期望值
- 盈虧比
- 凱利比例
- 風險評分

### 5. 📜 交易日誌

**路徑**: 自動交易 → 📜 交易日誌

**功能**:

#### 篩選器（側邊欄）

- **時間範圍**: 選擇起始/結束日期
- **交易對**: 多選過濾
- **交易類型**: 全部/開倉/平倉
- **損益**: 全部/獲利/虧損

#### 績效統計

| 指標 | 說明 |
|------|------|
| 總交易數 | 交易總次數 |
| 總損益 | 累計損益金額 |
| 勝率 | 獲利交易比例 |
| 平均獲利 | 獲利交易平均金額 |
| 平均虧損 | 虧損交易平均金額 |
| 盈虧比 | 平均獲利/平均虧損 |

#### 圖表分析

**損益趨勢**:
- 累積損益線圖
- 單筆損益柱狀圖

**收益分佈**:
- 損益分佈直方圖
- 分佈統計數據

**分類統計**:
- 按交易對統計
- 按策略統計
- 按方向統計（多頭/空頭）

#### 交易明細表格

**欄位**:
- 時間
- 交易對
- 動作（開倉/平倉）
- 方向（多頭/空頭）
- 價格
- 損益
- 損益%
- 手續費
- 原因

**功能**:
- 顏色標記（綠色獲利/紅色虧損）
- 匯出 CSV

## 💡 使用建議

### 新手流程

1. **學習階段**:
   - 使用風險管理工具熟悉計算
   - 查看交易日誌了解歷史表現

2. **測試階段**:
   - 在策略配置中建立測試策略
   - 使用測試網絡（Sandbox）
   - 設定保守型風險參數

3. **監控階段**:
   - 在儀表板查看整體績效
   - 在持倉監控查看即時狀態
   - 定期查看交易日誌

4. **優化階段**:
   - 根據績效調整策略參數
   - 使用風險分析工具評估
   - 逐步優化風險參數

### 風險 preset 建議

| 類型 | 每筆風險 | 停損 | 停利 | 槓桿 | 最大持倉 |
|------|----------|------|------|------|----------|
| **保守型** | 0.5-1% | 1-1.5% | 2-3% | 1-2x | 2-3 |
| **穩健型** | 1-2% | 2-3% | 4-6% | 3-5x | 3-5 |
| **積極型** | 2-3% | 3-5% | 6-10% | 5-10x | 5-8 |
