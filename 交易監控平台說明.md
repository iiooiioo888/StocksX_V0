# StocksX 交易監控平台 v4.0 - 三盤合一整合版

## 📋 功能概述

本次更新將原有的**策略訂閱（監控）**、**自動交易**和**模擬盤**功能整合為單一頁面，提供統一的交易管理界面。

## 🎯 核心功能

### 1. 交易模式切換
- **🟢 模擬盤模式**：虛擬資金交易，用於測試策略和熟悉系統
- **🔴 實盤模式**：真實資金交易，涉及實際風險
- **一鍵切換**：頂部狀態列可快速切換交易模式
- **模式標識**：每個策略卡片都顯示當前模式標識

### 2. 策略訂閱與監控（Tab 0）
- **即時價格**：批量加載訂閱策略的實時價格
- **即時信號**：自動計算並顯示交易信號（BUY/SELL/HOLD）
- **損益計算**：實時計算持倉損益（多頭/空頭）
- **手動操作**：
  - 🟢 做多：開多倉或反轉空倉
  - 🔴 做空：開空倉或反轉多倉
  - 📤 平倉：了結當前持倉
  - ❌ 移除：刪除策略訂閱

### 3. 新增訂閱（Tab 1）
- **市場選擇**：支援加密貨幣和傳統市場
- **產品分類**：按分類篩選產品
- **策略配置**：
  - 15+ 種專業策略可選
  - 時間框架選擇（5m/15m/1h/4h/1d）
- **資金設定**：
  - 初始資金配置
  - 槓桿倍數設定
  - 手續費率設定

### 4. 自動交易配置（Tab 2）
- **啟用控制**：開關控制自動交易啟用/停用
- **策略選擇**：選擇自動執行的策略
- **參數配置**：根據策略配置參數
- **風險控制**：
  - 停損/停利設定
  - 最大持倉比例
  - 每日最大虧損
  - 每日最大交易次數
- **通知設定**：
  - Bark 通知
  - Email 通知

### 5. 交易日誌（Tab 3）
- **篩選功能**：
  - 按模式篩選（全部/模擬盤/實盤）
  - 按類型篩選（全部/開倉/平倉）
  - 自定義顯示筆數
- **詳細記錄**：
  - 交易時間
  - 交易對
  - 交易類型
  - 持倉方向
  - 成交價格
  - 損益百分比
  - 交易原因

### 6. 績效分析（Tab 4）
- **績效指標**：
  - 總交易數
  - 勝/負場次
  - 勝率
  - 累計 P&L
  - 最佳/最差交易
- **可視化圖表**：
  - 累計損益走勢圖
  - 支援 Plotly 互動圖表

## 🛡️ 風險控制

- **停止所有交易**：立即平倉所有持倉

### 2. 持倉查詢
- **快速查詢**：輸入交易對查詢持倉
- **持倉信息**：方向、進場價、損益

### 3. 風險提示

用戶操作 → UI 層 → 業務邏輯層 → 數據層

## 🔧 技術架構

### 前端
- **Streamlit**：UI 框架
- **Plotly**：圖表可視化
- **Pandas**：數據處理

### 後端
- **SQLite**：用戶數據、持倉記錄
- **Celery**：異步任務（自動交易）
- **CCXT**：交易所 API 對接

### 數據流
1. **策略訂閱**：用戶添加訂閱 → 寫入 watchlist
2. **價格獲取**：批量調用 API → 快取 → 前端顯示
3. **信號計算**：基於策略算法 → 實時計算 → 前端顯示
4. **交易執行**：手動/自動 → 寫入 trade_log → 更新持倉

## 📝 使用流程

### 新手入門
1. **登入系統** → 進入交易監控頁面
2. **切換到模擬盤** → 熟悉界面和操作
3. **新增訂閱** → 選擇交易對和策略
4. **查看信號** → 觀察即時信號變化
5. **手動操作** → 嘗試做多/做空/平倉
6. **查看日誌** → 回顧交易記錄
7. **績效分析** → 評估策略表現

### 進階使用
1. **啟用自動交易** → 配置策略參數
2. **設定風險控制** → 停損/停利/持倉限制
3. **切換到實盤** → 謹慎進行真實交易
4. **監控持倉** → 實時追蹤損益
5. **查看報告** → 每日績效報告

## ⚠️ 風險提示

1. **模擬盤 ≠ 實盤**：模擬盤表現不代表實盤結果
2. **槓桿風險**：高槓桿可能導致資金快速損失
3. **策略風險**：歷史績效不代表未來表現
4. **市場風險**：加密貨幣波動劇烈，請謹慎投資
5. **技術風險**：網絡延遲、API 故障等可能影響交易
