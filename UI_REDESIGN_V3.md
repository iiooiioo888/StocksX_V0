## 🎨 視覺設計細節

### 色彩方案

| 用途 | 顏色 |
|------|------|
| 主背景 | `linear-gradient(160deg, #0a0a12, #12121f, #0f1724)` |
| Hero Banner | `linear-gradient(135deg, #1a1a2e, #16213e, #0f3460)` |
| 卡片背景 | `linear-gradient(135deg, #1e1e3a, #252545)` |
| 邊框 | `#2a2a4a`, `#3a3a5c` |
| 強調色 | `#6ea8fe` (藍色) |
| 文字主色 | `#e0e0e8` |
| 文字次要 | `#94a3b8` |

### 動畫效果

1. **Hero Banner 脈衝**
   ```css
   @keyframes pulse {
     0%, 100% { transform: scale(1); opacity: 0.5; }
     50% { transform: scale(1.1); opacity: 0.8; }
   }
   ```

2. **卡片 Hover**
   ```css
   .feature-card:hover {
     transform: translateY(-5px);
     box-shadow: 0 10px 30px rgba(110,168,254,0.2);
   }
   ```

3. **指標卡片載入**
   ```css
   @keyframes fadeInUp {
     from { opacity: 0; transform: translateY(20px); }
     to { opacity: 1; transform: translateY(0); }
   }
   ```

## ⚠️ 注意事項

1. **情緒數據依賴**：
   - 恐懼貪婪指數：`api_hub.get_current_fear_greed()`
   - VIX 指數：`api_hub.fetch_cboe_vix()`
   - 如 API 失敗會顯示「數據不可用」

2. **性能優化**：
   - 市場數據快取 120 秒
   - 可使用 `st.cache_data` 優化

3. **瀏覽器兼容**：
   - 現代瀏覽器（Chrome、Firefox、Safari、Edge）
   - IE 不支援（CSS 漸變、動畫）
