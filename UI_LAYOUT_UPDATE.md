# 界面佈局更新說明

## 📋 更新概述

將原本的 **層級式 Tab 佈局** 改為 **左側導航 + 頂部篩選** 的現代化佈局，避免信息過載，提升用戶體驗。

---

## 🎨 新佈局架構

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo] 📊 StocksX    [🔍 搜索代幣/代碼...]    [👤 用戶]   │  ← 頂部導航欄
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│  左側    │              主內容區                            │
│  導航    │  ┌────────────────────────────────────────┐     │
│          │  │  頂部二級篩選 (Tab): [現貨] [期貨] [期權] │     │
│  🏠 總覽 │  ├────────────────────────────────────────┤     │
│  🪙 加密 │  │  🔽 板塊篩選 (Dropdown)                │     │
│  📈 傳統 │  │  ├─ 🔥 熱門 / 主流                    │     │
│  💱 外匯 │  │  ├─ 🌐 DeFi / L2                      │     │

│          │  └────────────────────────────────────────┘     │

```

---

## ✅ 關鍵改進

### 1. 剝離「期貨」
- **舊版問題**：期貨作為獨立大類，與資產類別並列，造成歸屬混亂
- **新版方案**：期貨降級為**全局篩選條件 (Tab)**，歸屬到對應資產下
  - 股指期貨 → 傳統股票/指數
  - 商品期貨 → 大宗商品
  - 外匯期貨 → 外匯市場

### 2. 昇華「情緒」
- **舊版問題**：情緒指標作為普通行情列表，重要性被稀釋
- **新版方案**：
  - 在**總覽儀表板**中獨立展示為 **情緒儀表板 Widget**
  - 視覺化設計：大字體 + 顏色編碼（綠色=貪婪，紅色=恐懼）
  - 支援恐懼貪婪指數 & VIX 波動率指數

### 3. 統一搜索
- **全局搜索框**：位於頂部導航欄中央
- **未來規劃**：輸入 `BTC` 同時顯示現貨/期貨，輸入 `Gold` 顯示現貨/期貨/相關股票

## 🎨 視覺設計亮點

### 1. 指標卡片增強
- **Hover 動畫**：鼠標懸停時卡片上浮 + 陰影加深
- **漸變背景**：`linear-gradient(135deg, #1e1e3a, #252545)`
- **邊框高亮**：`#3a3a5c` 低調邊框

### 2. 情緒儀表板
- **視覺中心**：2.5rem 大字體顯示數值
- **顏色編碼**：
  - 恐懼貪婪 > 50：綠色 `#00ff00`
  - 恐懼貪婪 ≤ 50：紅色 `#ff4444`
  - VIX < 20：綠色
  - VIX > 30：紅色
  - VIX 20-30：黃色

### 3. 按鈕交互
- **Active 狀態**：漸變藍色背景 `linear-gradient(135deg, #4a6cf7, #6366f1)`
- **Hover 狀態**：邊框高亮 + 背景色變化

## 🚀 使用指南

### 首次啟動
```bash
streamlit run app.py
```

### 導航流程
1. **左側導航**：選擇資產大類（如「🪙 加密資產」）
2. **頂部 Tab**：選擇交易類型（如「現貨」或「期貨/合約」）
3. **Dropdown**：選擇板塊（如「🔥 主流」或「🌐 DeFi/L2」）

### 總覽儀表板
- 點擊「🏠 總覽儀表板」查看：
  - 期貨報價 & 熱門標的
  - 情緒儀表板（恐懼貪婪 + VIX）
  - 快速導航（加密回測、傳統回測、策略監控、歷史）
