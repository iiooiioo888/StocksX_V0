## 🚀 使用方式

### 1. 启动应用
```bash
cd StocksX_V0
streamlit run app.py
```

### 2. 访问 AI 策略中心
- 首页 → 点击 **🧠 AI 策略中心** 卡片
- 或直接访问：`http://localhost:8501/9_🧠_AI 策略`

### 3. 使用策略回测对比
- 首页 → 点击 **📈 策略回测对比** 卡片
- 选择要对比的策略（支持多选）
- 设置时间范围和初始资金
- 点击 **🚀 运行回测对比**
