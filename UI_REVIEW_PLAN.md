## 🎯 優化路線圖

### Phase 1: 核心功能優化（1-2 週）

1. **全局搜索功能**
   - 支援交易對、策略、歷史記錄搜索
   - 快速跳轉到對應頁面

2. **回測非同步化**
   - 整合 Celery 任務隊列
   - 加入進度條、任務狀態查詢

3. **歷史記錄增強**
   - 分頁、篩選、排序
   - 多筆對比功能

4. **即時監控升級**
   - WebSocket 即時推送
   - 信號歷史記錄

### Phase 2: 用戶體驗優化（2-3 週）

5. **策略參數管理**
   - 可展開參數設定
   - 儲存/載入參數預設

6. **數據可視化增強**
   - 更多績效指標圖表
   - 策略對比視覺化

7. **快捷操作**
   - 常用回測快捷鍵
   - 自訂快捷方式

### Phase 3: 進階功能（3-4 週）

8. **財報整合**
   - 整合 FMP API
   - 顯示基本面數據

9. **風險管理**
   - 曝險計算
   - VaR、壓力測試

10. **權限管理**
    - RBAC 權限分級
    - 操作審計日誌
