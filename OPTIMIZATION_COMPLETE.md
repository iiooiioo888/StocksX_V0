### 5. WebSocket 即時推送整合 ✅

**新增檔案**: 
- `src/websocket_server.py` (FastAPI WebSocket 服務)
- `pages/8_⚡_即時監控.py` (Streamlit 即時監控頁面)

**功能說明**:

#### WebSocket 服務端 (FastAPI)

**端點**:
```
ws://localhost:8001/ws/{symbol}?token=JWT_TOKEN
```

**功能**:
- ✅ JWT 令牌驗證
- ✅ 訂閱管理（subscribe/unsubscribe）
- ✅ 即時價格推送（每秒）
- ✅ 交易信號推送（每 5 秒）
- ✅ 連接管理器
- ✅ 健康檢查端點

**推送數據格式**:
```json
{
  "type": "price_update",
  "data": {
    "symbol": "BTC/USDT",
    "price": 67500.00,
    "change_pct": 2.5,
    "timestamp": 1709467200000
  }
}
```

```json
{
  "type": "signal",
  "data": {
    "symbol": "BTC/USDT",
    "action": "BUY",
    "strategy": "sma_cross",
    "confidence": 0.85,
    "price": 67500.00,
    "timestamp": 1709467200000
  }
}
```

#### 即時監控頁面 (Streamlit)

**功能**:
- ✅ 訂閱管理側邊欄
- ✅ WebSocket 連接/斷開
- ✅ 即時價格卡片（3 秒刷新）
- ✅ 最新信號列表
- ✅ 持倉監控
- ✅ 損益計算

**頁面結構**:
```
📡 即時監控
├── 📊 即時價格（多交易對卡片）
├── 🔔 最新信號（BUY/SELL 信號）
└── 📈 持倉監控（損益、操作）
```

**使用方式**:
```
1. 啟動 WebSocket 服務：
   python -m src.websocket_server

2. 開啟即時監控頁面：
   streamlit run pages/8_⚡_即時監控.py

3. 點擊「🔌 連接 WebSocket」
4. 在側邊欄管理訂閱
5. 查看即時價格和信號
```
