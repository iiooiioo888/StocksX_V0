# 安全策略

## 報告安全漏洞

如果你發現 StocksX 的安全漏洞，請**不要**在公開的 GitHub Issue 中報告。

請通過以下方式聯繫：
- 發送郵件至：security@stocksx.dev（如有）
- 或在 GitHub 上建立私人安全報告

我們會在收到報告後 **48 小時內** 回覆，並在 **7 天內** 提供修復方案。

## 安全措施

- 🔒 密碼：PBKDF2-SHA256（100,000 iterations）+ 隨機 salt
- 🔑 認證：JWT 令牌，1 小時超時
- 🛡️ 限流：令牌桶演算法，登入鎖定機制
- 📝 審計：完整登入日誌
- 🔍 CI：bandit 靜態安全分析 + safety 依賴漏洞掃描
- 🐳 Docker：非 root 使用者運行、tini init 系統
- 🔄 Dependabot：自動依賴安全更新
