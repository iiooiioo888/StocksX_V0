## ⚠️ 需特殊數據的策略（10 個）

以下策略需要 tick 級數據或訂單簿數據才能進行有效回測：

| 策略 | 數據需求 | 當前限制 | 解決方案 |
|------|---------|---------|---------|
| vpin | 逐筆成交數據 | 日線數據無法計算 VPIN | 購買 Tushare Pro tick 數據 |
| level2 | 訂單簿數據 | 缺少 Level 2 深度 | 使用券商歷史訂單簿 |
| order_flow | 逐筆成交 | 無法識別主動買賣 | 使用 tick 數據源 |
| cum_delta | 逐筆成交 | Delta 計算不準確 | 升級數據源 |
| poc_va | 訂單簿 + 成交量分佈 | 無法計算 POC | 使用 futures 數據 |
| iceberg | 逐筆訂單 | 冰山訂單無法識別 | 需要 L3 數據 |
| kyle_lambda | 逐筆成交 + 價格 | Lambda 估計不準確 | tick 數據 |
| tick_rule | 逐筆成交 | 無法應用 Tick Rule | tick 數據 |
| quote_stuffing | 訂單簿 + 時間戳 | 無法偵測 Quote Stuffing | L3 數據 |
| micro_price | 訂單簿最佳買賣價 | 微價格計算不準確 | L2 數據 |

**建議優先級**:
1. 🔴 高：vpin, level2, order_flow（對微結構策略至關重要）
2. 🟡 中：cum_delta, poc_va, iceberg
3. 🟢 低：kyle_lambda, tick_rule, quote_stuffing, micro_price

- **A 股**: Tushare Pro (https://tushare.pro)

## 📝 回測方法論

### 數據設置
- **期間**: 2020-01-01 至 2023-12-31（3 年）
- **頻率**: 日線（微結構策略除外）
- **標的**: A 股主要成分股（000001.SZ 等）
- **調整**: 前復權

### 回測參數
- **初始資金**: ¥100,000
- **手續費率**: 0.1%（雙邊）
- **滑點**: 0.1%
- **倉位管理**: 固定比例 / Kelly / 波動率目標

### 評估指標
- **Sharpe 比率**: 年化收益率 / 年化波動率 × √252
- **Calmar 比率**: 年化收益率 / 最大回撤
- **Sortino 比率**: 年化收益率 / 下行波動率 × √252
- **最大回撤**: (組合最低點 - 前期最高點) / 前期最高點
- **總回報**: (最終資金 - 初始資金) / 初始資金

### 通過標準
- **Sharpe > 0.5**: 通過 ✅
- **Sharpe 0.3-0.5**: 需優化 ⚠️
- **Sharpe < 0.3**: 需重新設計 ❌
