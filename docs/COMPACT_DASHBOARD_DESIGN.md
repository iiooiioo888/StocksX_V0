## 🔧 核心功能模組

### 1. **三視圖切換系統**

#### 儀表板視圖（Dashboard）
- **用途**: 快速概覽所有持倉
- **佈局**: 3 列卡片網格
- **信息密度**: 中等
- **適用場景**: 日常監控

#### 網格視圖（Grid）
- **用途**: 詳細數據對比
- **佈局**: 緊湊型表格
- **信息密度**: 高
- **適用場景**: 數據分析

#### 詳細視圖（Detail）
- **用途**: 深入查看單個持倉
- **佈局**: 展開式列表
- **信息密度**: 最高
- **適用場景**: 問題診斷

### 2. **分層快取系統**

```
L1: 即時價格（3 秒 TTL）
     ↓
L2: 交易信號（15 秒 TTL）
     ↓
L3: 用戶數據（30-120 秒 TTL）
     ↓
L4: 純函數快取（永久）
```

### 3. **實時數據流**

```python
# 批量加載價格
prices = get_prices_l1(symbols_tuple)

# 批量計算信號
signals = get_signals_l2(watchlist_tuple)

# 自動刷新（可配置間隔）
if auto_refresh:
    time.sleep(interval)
    st.rerun()
```

### 4. **快速操作系統**

#### 一鍵交易
- 🟢 **做多**: 開多倉或反轉空倉
- 🔴 **做空**: 開空倉或反轉多倉
- **平**: 平掉當前持倉

#### 緊急控制
- 🛑 **緊急停止**: 立即平倉所有持倉（實盤專屬）

#### 持倉查詢
- 輸入交易對 → 查詢持倉詳情

## 🎯 優化技術

### 1. **CSS 性能優化**
```css
/* 硬件加速動畫 */
.metric-card:hover {
    transform: translateY(-2px);  /* 使用 transform 而非 position */
    will-change: transform;        /* 提示瀏覽器優化 */
}

/* 減少重排 */
.signal-badge {
    display: inline-flex;          /* 避免 block 元素的重排 */
    align-items: center;
}
```

### 2. **響應式設計**
```css
/* 超寬屏（≥2000px） */
@media (min-width: 2000px) {
    .data-grid {
        grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    }
}

/* 移動端（≤768px） */
@media (max-width: 768px) {
    .data-grid {
        grid-template-columns: 1fr;
    }
}
```

### 3. **加載優化**
- **骨架屏動畫**: 加載時顯示閃爍效果
- **延遲加載**: 只加載當前視圖需要的數據
- **快取命中**: L1/L2/L3 多級快取減少 API 調用

## 🛠️ 使用指南

### 1. **啟動緊湊型儀表板**
```bash
streamlit run pages/5_📡_交易監控_v6_compact.py
```

### 2. **切換視圖模式**
1. 點擊側邊欄「視圖模式」
2. 選擇：
   - **儀表板**: 快速概覽
   - **網格視圖**: 數據對比
   - **詳細視圖**: 深入分析

### 3. **配置自動刷新**
1. 側邊欄勾選「自動刷新」
2. 調整間隔滑塊（3-60 秒）
3. 點擊「🔄」按鈕手動刷新

### 4. **快速交易操作**
```
持倉卡片上的按鈕：
[🟢] = 做多 / 反轉空倉
[🔴] = 做空 / 反轉多倉
[平] = 平掉持倉
```
